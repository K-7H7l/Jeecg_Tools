## 介绍

**本工具为jeecg框架漏洞利用工具非jeecg-boot！**

**包含poc：**

- **登录绕过检测**
- **jeecgFormDemo文件上传**
- **common文件上传**
- **icon文件上传**
- **Xstream反序列化**



## 使用

**运行于jdk8环境**

```shell
java -jar Jeecg_Tools-1.0-java8.jar
```
